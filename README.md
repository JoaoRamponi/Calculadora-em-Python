# 🧮 Calculadora em Python

Este projeto consiste em uma **calculadora simples desenvolvida em Python**, executada diretamente no terminal.
O programa permite que o usuário escolha diferentes operações matemáticas e receba o resultado imediatamente.

O sistema utiliza **estrutura de repetição (`while`)** e **estrutura condicional (`match case`)** para controlar o fluxo do programa.

---

# 📌 Funcionalidades

A calculadora possui as seguintes operações:

* ➕ **Soma**
* ➖ **Subtração**
* ✖ **Multiplicação**
* ➗ **Divisão**
* 🔢 **Contador de 0 a 100**
* ❌ **Encerrar o programa**

---

# 🛠 Tecnologias Utilizadas

* Python 3
* Estruturas utilizadas:

  * `while`
  * `match case`
  * `for`
  * `input()`
  * `print()`

---

# ▶️ Como Executar o Projeto

1. Certifique-se de ter o **Python instalado** no computador.

2. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

3. Entre na pasta do projeto:

```bash
cd nome-do-projeto
```

4. Execute o programa:

```bash
python calculadora.py
```

---

# 📋 Como o Programa Funciona

1. O usuário informa **dois números**.
2. Escolhe a operação desejada no menu.
3. O programa executa o cálculo e mostra o resultado.
4. O menu continua aparecendo até que o usuário escolha **encerrar o programa**.

---

# 📄 Exemplo de Uso

```
Bem vindo a Calculadora em Python
Digite o primeiro numero: 10

Selecione a operação que voce deseja fazer
1 = Soma
```
